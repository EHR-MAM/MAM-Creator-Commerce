Marketing · Signup
Join — Creator & Vendor Application
sensedirector.com/mam/join
Open Live ↗
Who uses it: Creators applying for a store; vendors applying to list products. Design: Two-column layout — creator form on left (gold), vendor form on right (orange). Both forms submit via WhatsApp to the ops team. No account is created automatically — ops provisions accounts manually through the Admin Dashboard.
On creating new accounts: The Join page collects applications via WhatsApp. Accounts are provisioned manually: Admin Dashboard → Creators tab → add creator (sets email, password, handle → store URL auto-assigned as /mam/{handle}). The API endpoint is POST /mam-api/auth/register with role=influencer or role=vendor. A self-serve signup flow can be built in a future sprint if volume warrants it.
